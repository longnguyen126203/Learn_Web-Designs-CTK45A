HTML và CSS là gì?

HTML là chữ viết tắt của HyoerText Markup Language. Nó giúp người dùng tạo và cấu trúc thành phần trong trang web hoặc ứng dụng, phân chia các đoạn văn, links, blockquotes, vv...,

CSS là ngôn ngữ tạo phong cách cho trang web - Cascarding Style Sheet language. Nó dùng để tạo phong cách và định kiểu cho những yếu tố được viết dưới dạng ngôn ngữ đánh đấu, như là HTML.
